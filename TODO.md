# 🗺️ AEGIS-VORTEX : FUTURE_GRID (TODO) 🏎️⚡🟢

L'aventure AegisVortex ne fait que commencer. Voici la feuille de route des évolutions prévues pour transformer ce bouclier en une forteresse numérique universelle.

---

## 🚀 Priorité : Distribution & Accessibilité
- [ ] **📦 Standalone Executable (.exe) :** Compilation du projet via *PyInstaller* ou *Nuitka* pour permettre un lancement sans installation de Python.
- [ ] **🌐 Localisation (Multi-langue) :** Support natif de l'Anglais (EN) et de l'Espagnol (ES) pour toucher une audience internationale.
- [ ] **🐍 PyPI Package :** Publication du moteur `secu_files` sur PyPI pour permettre à d'autres développeurs d'utiliser la puissance d'AegisVortex dans leurs projets.

---

## 🖱️ Priorité : Ergonomie & UX
- [ ] **🪐 Intégration Contextuelle Windows :** Ajout d'une option "Chiffrer avec AegisVortex" au clic droit dans l'Explorateur de fichiers.
- [ ] **📂 File Queue (File d'attente) :** Système de traitement par lots pour chiffrer/déchiffrer plusieurs dossiers de manière séquentielle.
- [ ] **♻️ Smart-Wipe (Nettoyage Sécurisé) :** Option pour écraser les données originales avec des données aléatoires (Standard DoD 5220.22-M) avant la suppression pour empêcher toute récupération.

---

## 🏎️ Priorité : Performance & Core
- [ ] **📊 Module de Benchmark :** Outils d'analyse de vitesse disque pour conseiller automatiquement le niveau de compression idéal à l'utilisateur.
- [ ] **☁️ Cloud-Bridge API :** Possibilité d'envoyer directement les archives chiffrées vers des services S3 (AWS) ou MEGA après traitement.
- [ ] **🗝️ Support Hardware (Yubikey) :** Utilisation de clés physiques 2FA pour déverrouiller la Vault sans mot de passe manuel.

---

## 🛠️ Maintenance & Qualité
- [ ] **🧪 Tests Unitaires :** Implémentation de `pytest` pour garantir la stabilité du moteur AEC à chaque mise à jour.
- [ ] **📜 Documentation API :** Génération automatique de la doc via Sphinx pour le moteur de chiffrement.
